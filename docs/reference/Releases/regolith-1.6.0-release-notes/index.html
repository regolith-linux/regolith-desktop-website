    1.6 Release Notes

    Regolith R1.6 is a feature release which includes several improvements and optimizations. Regolith 1.6’s primary feature is a return of the ISO installation method.

    Features #

    FeatureScreenshotDescription
    Return of Regolith Linux ISOUp to Regolith 1.4.1, we hand-built each ISO for major version releases. This became too brittle and time intensive, and so work began on build automation to generate the ISO. This took quite awhile to complete but is now available for the 1.6 release.
    New Look: MidnightFabian (naib864) has provided a distinctive new look with new font, icons, and GTK theme.
    New Look: Solarized LightRicardo PÃ©rez (ricpelo) has provided a solarized light theme.
    New Status Indicator: app-launcherRegolith user Laurent provided the idea to add a status indicator to launch Rofi, and so i3xrocks-app-launcher was added for those that prefer to use the mouse to launch apps.
    Upgrades to i3-gaps and Rofii3-gaps is updated to 4.19.1 and Rofi to 1.6.1.

    Fixes #

    Have a look at the R1.6 project page for a list of bug fixes.
