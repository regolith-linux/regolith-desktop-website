    Change the Terminal Font Size
    NOTICE: This page was copied from the Regolith 1.x website and has not been updated for Regolith 2. It may contain out of date information.
    1. Create or add the following value to your ~/.config/regolith2/Xresources file:
    gnome.terminal.font:	JetBrains Mono 14
    1. Reload the Xresources configuration:
    $ regolith-look refresh
    1. Open a new terminal to see the change take effect.
